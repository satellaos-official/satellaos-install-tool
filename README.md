## 📌 Neden SatellaOS? / Why SatellaOS?

| | 🇹🇷 Türkçe | 🇬🇧 English |
|---|---|---|
| 🏗️ **Altyapı / Infrastructure** | Debian Netinst altyapısı üzerine kuruludur. Üç temel hedef: Minimalizm, Modernlik, Stabilite. | Built on Debian Netinst infrastructure. Three core goals: Minimalism, Modernity, Stability. |
| 📦 **Paket Sayısı / Package Count** | Çoğu Debian/Ubuntu XFCE dağıtımı 2000+ paketle gelir. SatellaOS kurulum tercihlerine göre yalnızca 1000–1600 dpkg paketi içerir. | Most Debian/Ubuntu XFCE distros ship with 2000+ packages. SatellaOS includes only 1000–1600 dpkg packages depending on your installation choices. |
| 🎨 **Arayüz / Interface** | Eski görünümlü XFCE dağıtımlarının aksine modern bir arayüz sunar. | Offers a modern interface unlike the outdated look of most XFCE distributions. |
| 🚫 **Bloatware** | Kullanıcı özgürlüğünü ön planda tutar; sistem bloatware'den tamamen arındırılmıştır. | Prioritizes user freedom; the system is completely free of bloatware. |
| 🧩 **Modülerlik / Modularity** | Tamamen modülerdir: `SatellaOS-program-installer-tool.sh` ile 30 farklı programı tercihinize göre kurabilirsiniz. | Fully modular: install 30 different programs based on your preferences using `SatellaOS-program-installer-tool.sh`. |

## ⚠️ Uyarılar / Warnings

| | 🇹🇷 Türkçe | 🇬🇧 English |
|---|---|---|
| 🖐️ **Dokunmatik Panel / Touchscreen** | SatellaOS dokunmatik paneller için tasarlanmamıştır; elimizde dokunmatik ekran donanımı bulunmamaktadır. | SatellaOS is not designed for touch panels, as we currently do not have touchscreen hardware available. |
| 🎮 **Oyunculuk / Gaming** | SatellaOS oyuncular için tasarlanmamıştır; Proton gelişmiş olsa da genel oyun performansı açısından Windows hâlâ üstündür. | SatellaOS is not intended for gamers; although Proton has improved significantly, Windows still offers better overall gaming performance. |
